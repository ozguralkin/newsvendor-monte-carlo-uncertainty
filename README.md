# Monte Carlo Sensitivity Analysis for the Newsvendor Model

This project studies the impact of parameter uncertainty on the classical Newsvendor problem under normally distributed demand.

Using Monte Carlo simulation, we analyze how estimation errors in the demand distribution parameters and uncertainty in the salvage price affect the quality of the ordering decision and the resulting expected profit.

The project investigates:
- the effect of historical sample size on parameter estimation,
- uncertainty in the salvage price,
- the impact of underage and overage cost asymmetry,
- the role of demand variability.

The analysis is based on the critical fractile solution of the Newsvendor model and evaluates performance degradation through regret and relative expected-profit metrics.

Implemented in MATLAB for the course:
**Ottimizzazione e simulazione in condizioni di incertezza**
(Politecnico di Torino).
